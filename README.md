# Block
https://ka-rui.github.io/Block/
A productivity management website that executes all distraction

The new world is full of distraction and block is there to "block" it. Haha.

Anyway, there are several features in this website:
1. Switch between dark mode and light mode
2. Built-in watch and timer inside
3. Classify tasks into high, medium, and low priority
4. Track your achievements with the visualization of progress bar

This website concept is simplistic and elegant with the purpose of removing any distraction that most of "productivity" apps have nowadays.

DISCLAIMER: Mobile won't work as it is engineered for desktop purposes (I might provide mobile version in the future).
